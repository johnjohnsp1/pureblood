# Pure Blood v1

> A Penetration Testing Framework created for Hackers / Pentester / Bug Hunter

## Menu
* Web Pentest <br>
 | Banner Grab <br>
 | Whois <br>
 | Traceroute <br>
 | DNS Record <br>
 | Reverse DNS Lookup <br>
 | Zone Transfer Lookup <br>
 | Port Scan <br>
 | Admin Panel Scan <br>
 | Subdomain Scan <br>
 | CMS Identify <br>
 | Reverse IP Lookup <br>
 | Subnet Lookup <br>
 | Extract Page Links
* Generator <br>
 | Deface Page

## Installation

> Any Python Version.

```
$ git clone https://github.com/cr4shcod3/pureblood
$ cd pureblood
$ pip install -r requirements.txt
```
